## 專案概要
本專案旨在開發一個簡易的投資策略模擬工具。

## 主要功能
1. 設定一些參數讓使用者能夠調整形成自己的策略，例如移動平均線的天數、買賣點的條件等。
2. 提供一個簡單的介面讓使用者能夠選擇不同的股票代號，並查看該股票的歷史價格走勢圖。 (我希望可以選台灣幾檔熱門股票)
3. 能夠在選擇完股票的時間與設定完策略後，進行爬蟲爬取那段時間股票的走勢，並畫在結果圖上類似真正股市股票的那種柱狀(折線)圖
4. 並進行策略分析，跟單純持有進行比較(所以會有三種圖疊在一起，一個是原始股市股票的柱狀圖，一個是策略買賣點的K線，另一個是單純持有的K線)，並用不同顏色區分。
5. 有不同的用戶帳號，可以存不同用戶的策略與模擬結果 (用PostgreSQL實現)。

## 主要架構

### 前端 (簡潔，不要太花俏)
1. React.js
2. 希望可以有Dark Theme (可以採用一些寫好的UI Package)
3. 視覺Chart
4. 使用者帳號系統
5. 策略參數設定介面
6. 結果顯示介面
7. 可以適當的有一些動畫

### 後端
1. 架構: FastAPI (python)
2. 資料庫: PostgreSQL
3. 存User Data
4. 爬蟲: 爬取股票歷史資料 (可以用yfinance或其他API)
5. 資料分析: 使用pandas進行資料處理與分析
6. API設計: 提供前端所需的資料
